> 版本 7.40b | 最终目标：写一篇说明 Dota2 底层数据逻辑的文章

---

## 模块划分

| Tab | 主题 | 内容 |
|-----|------|------|
| `#economy` | 经济数据 | 兵线、工资、野怪、赏金神符 |
| `#buildings` | 建筑数据 | 防御塔属性（血量、护甲、攻击力、攻速） |
| `#resources` | 战略资源 | 莲花池、痛苦魔方、经验神龛、强化神符、肉山 |

---

## 一、经济数据 (`#economy`)

### 1.1 兵线经济

| 数据 | 说明 |
|------|------|
| 基础属性 | 近战/远程/旗手/攻城车的金币、经验 |
| 升级公式 | 每 7.5 分钟升级，金币和经验增长 |
| 波次收益 | 单波金币/经验（随时间变化曲线） |
| 补刀价值 | 完美补刀 vs 漏刀的经济差距 |

**数据源**: `creeps.json`

### 1.2 被动工资 (GPM)

| 数据 | 说明 |
|------|------|
| 基础工资 | 每秒/每分钟金币 |
| 公式 | `85 + 6×分钟` (基础 + 递增) |

**数据源**: 需补充

### 1.3 野怪经济

| 数据 | 说明 |
|------|------|
| 营地分类 | 小野/中野/大野/远古野 |
| 单营收益 | 金币范围、经验值 |
| 刷野效率 | 每分钟理论最大收益 |
| 叠野收益 | 叠 1-5 层的收益倍增 |

**数据源**: `neutrals.json`

### 1.4 赏金神符

| 数据 | 说明 |
|------|------|
| 刷新时间 | 0:00 起，每 3 分钟 |
| 金币公式 | `36 + 3×分钟` |
| 经验公式 | `35 + 5×分钟` |
| 分布机制 | 团队共享 vs 个人拾取 |

**数据源**: 需补充

## 二、建筑数据 (`#buildings`)

### 2.1 防御塔

| 等级 | 血量 | 护甲 | 攻击力 | 攻速 | 攻击距离 |
|------|------|------|--------|------|----------|
| T1 | 1800 | 12 | 88-92 | 0.9s | 700 |
| T2 | 2500 | 16 | 170-174 | 0.9s | 700 |
| T3 | 2500 | 16 | 170-174 | 0.9s | 700 |
| T4 | 2600 | 21 | 170-174 | 0.9s | 700 |

### 2.2 兵营

| 类型 | 血量 | 护甲 | 回血 |
|------|------|------|------|
| 近战兵营 | 2200 | 15 | 5/s |
| 远程兵营 | 1300 | 10 | 5/s |

### 2.3 遗迹 (Ancient)

| 属性 | 数值 |
|------|------|
| 血量 | 4500 |
| 护甲 | 23 |
| 回血 | 12/s |

### 2.4 机制

- **后门保护**: T2+ 减伤 50%，回血 90/s（遗迹 180/s）
- **塔护甲加成**: 每多 1 敌方英雄（超过 2 个）+3 护甲
- **防御符文**: 建筑和兵线无敌 6 秒

**数据源**: `buildings.json`

## 三、战略资源 (`#resources`)

### 3.1 肉山 (Roshan)

| 属性 | 数值 |
|------|------|
| 血量 | 6000 |
| 护甲 | 30 |
| 魔抗 | 55% |
| 攻击力 | 75 |
| 刷新 | 8-11 分钟 |
| 掉落 | 1st: 盾 / 2nd: +奶酪 / 3rd+: +刷新碎片或A杖 |

### 3.2 折磨者 (Tormentor)

| 属性 | 数值 |
|------|------|
| 刷新时间 | 首次 7:00，之后每 10 分钟 |
| 掉落 | 阿哈利姆神杖碎片 |
| 位置 | 两侧野区各一个 |

### 3.3 莲花池 (Lotus Pool)

| 属性 | 数值 |
|------|------|
| 位置 | 河道中央 |
| 效果 | 治疗 + 魔法恢复 |
| 刷新 | 每 3 分钟 |

### 3.4 经验神龛 (Wisdom Rune)

| 属性 | 数值 |
|------|------|
| 刷新时间 | 7:00 起，每 7 分钟 |
| 经验公式 | `280 × n` (n = 刷新次数) |
| 位置 | 双方野区各一个 |

### 3.5 强化神符 (Power Rune)

| 属性 | 数值 |
|------|------|
| 刷新时间 | 6:00 起，每 2 分钟 |
| 类型 | 双倍、加速、隐身、幻象、极速、奥术 |
| 位置 | 河道两侧 |

### 3.6 痛苦魔方 (Tormentor Cube)
