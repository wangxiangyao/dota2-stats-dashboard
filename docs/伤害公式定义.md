# 伤害公式定义

> 此文档用于讨论和定义每个技能的伤害计算公式
> 总技能数: 254
> 生成时间: 2026/1/19 13:20:08

## 公式类型说明

| 类型 | 说明 | 计算方式 |
|------|------|----------|
| simple | 简单直接伤害 | 直接读取数值 |
| dot | 持续伤害 | 每秒伤害 × 持续时间 |
| multi | 多段伤害 | 单次伤害 × 次数 |
| percent | 百分比伤害 | 基准值 × 百分比 |
| special | 特殊公式 | 需要自定义计算 |

## 字段说明

| 字段 | 说明 |
|------|------|
| formulaType | 公式类型 (simple/dot/multi/percent/special) |
| primaryKey | 主伤害字段（从 damageKeys 中选择） |
| formula | 计算公式表达式 |
| effectiveType | 等效伤害类型 (PURE/MAGICAL/PHYSICAL)，考虑配套机制后的实际类型 |
| calculatedDamage | 计算后的各等级伤害值 |
| notes | 备注说明 |
